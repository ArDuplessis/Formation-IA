# Projets de Formation IA

## Structure des Projets
Chaque projet est organisé de la manière suivante :
```
projet_nom/
  ├── src/            # Code source
  ├── tests/          # Tests unitaires
  ├── data/           # Données utilisées
  ├── docs/           # Documentation
  └── README.md       # Description et instructions
```
